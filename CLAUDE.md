# 프로젝트 가이드라인

## 프로젝트 개요

React-like 라이브러리를 직접 구현하는 학습 프로젝트.
VanJS 스타일 API(`const {div, p} = tags`)를 기반으로 한다.

## 구조

- `src/lib.js` — 핵심 구현 (단일 파일에 전체 흐름 유지)
- `src/lib.test.js` — 테스트
- `note/` — 학습 자료 및 구현 계획 정리
- `docs/` — 구현 단계별 상세 계획

## 테스트

- 모든 구현 또는 코드 변경 후, 반드시 테스트를 실행하여 정상 동작을 검증할 것:
  ```
  npx vitest run
  ```
- 모든 테스트가 통과할 때까지 작업을 완료로 간주하지 말 것.
- 테스트가 실패하면 문제를 수정하고 다시 테스트를 실행한 뒤 다음 작업으로 넘어갈 것.
